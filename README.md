# Newsletter Engine

Fetches tech articles from RSS feeds and uses Claude AI to generate a weekly newsletter.

## What It Does
- Fetches latest articles from multiple RSS feeds
- Sends all articles to Claude AI for curation
- Generates compelling newsletter with headline and editor note
- Picks top story and quick hits
- Sends formatted newsletter via email

## Sources
- AWS News Blog
- TechCrunch AI
- The Verge Tech

## Tech Stack
- Python 3
- Claude API (Anthropic)
- AWS SES
- feedparser
- boto3
